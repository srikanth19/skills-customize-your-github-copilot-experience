# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice working with strings and files in Python by building a small text-processing toolkit that reads a text file, analyzes its contents, and transforms text output.

## 📝 Tasks

### 🛠️ Read and Analyze Text Data

#### Description
Write a program that opens a text file, reads its contents, and reports useful statistics about the text.

#### Requirements
Completed program should:

- Open and read text from a file named `sample.txt`.
- Count and print the total number of characters, words, and lines.
- Display the top 5 most common words (case-insensitive).
- Ignore punctuation when counting words.

### 🛠️ Transform and Export Processed Text

#### Description
Add text-manipulation features that clean and transform the input, then save the transformed result to a new file.

#### Requirements
Completed program should:

- Convert all text to lowercase.
- Replace multiple spaces with a single space.
- Remove leading and trailing whitespace from each line.
- Save the processed text to `processed_output.txt`.
